# F.1 Science Final Exam MC Trainer

Static GitHub Pages version of the bilingual Form 1 Science MC trainer.

## Publish With GitHub Pages

1. Create a new public GitHub repository.
2. Upload everything in this `github-pages-ready` folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save and wait a few minutes.

Your public URL will usually be:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```

## Files

- `index.html`: main webpage
- `app.js`: question data and interaction logic
- `styles.css`: visual styling
- `assets/question-images/`: generated question images
